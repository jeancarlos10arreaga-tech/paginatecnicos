DISEÑO WEB





El Diseño Web es un módulo técnico que se enfoca en la creación, estructura y mantenimiento de sitios y aplicaciones web. En este módulo, los estudiantes aprenden a combinar la creatividad con los conocimientos técnicos para desarrollar páginas visualmente atractivas, interactivas y funcionales.
Se utilizan lenguajes como HTML, CSS y JavaScript para diseñar la apariencia y el comportamiento de las páginas. Además, se estudian principios de usabilidad, accesibilidad y experiencia de usuario (UX), que ayudan a crear sitios que sean fáciles de navegar y agradables para el visitante.
El diseño web también incluye el uso de herramientas gráficas y editores de código, así como la adaptación de los contenidos a diferentes dispositivos, como computadoras, tablets y teléfonos.
En conjunto, este módulo desarrolla la capacidad de transformar ideas en proyectos digitales reales, preparando al estudiante para el mundo laboral y para contribuir al desarrollo tecnológico a través del diseño de sitios modernos y eficientes.
